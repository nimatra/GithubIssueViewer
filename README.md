

# Github Issue Viewer

TypeScript/React/Redux

## npm scripts

### Install
$ npm install

### Dev
```bash
$ npm run dev:client
```

This runs a development mode server with live reload etc.

Open `http://localhost:8080/app` in your browser.
### Tests

#### Single Run
```bash
$ npm run test
```

#### Watch Files
```bash
$ npm run test:watch
```

#### Coverage
```bash
$ npm run cover
```
